
# Sweet Home Api

second year project of esi sba



## Installation

Clone the project

```bash
  git clone https://github.com/oussamabng/Sweet-Home-Esi
  cd Sweet-Home-Esi
```

install dependencies and run 

```bash
  npm install --save
  npm start
```
